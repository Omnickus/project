Возможности редактора


Заголовки

# (текст) - заголовок первого уровня, в HTML тег 'h1'

## (текст) - заголовок второго уровня, в HTML тег 'h2'

### (текст) - заголовок третьего уровня, в HTML тег 'h3'

#### (текст) - заголовок четвертого уровня, в HTML тег 'h4'

##### (текст) - заголовок пятого уровня, в HTML тег 'h5'

###### (текст) - заголовок шестого уровня, в HTML тег 'h6'

Горизонтальные линии

" * * * * * " , " ---------- " , " _ _ _ _ _ "
Можно использовать любой из предложенных вариантов, результат будет один и тот же.

Абзацы и параграфы

В маркдаун все блочные элементы (заголовки, списки, абзацы и т.д.) — все, что визуально начинается с новой строки отбивается пустой строкой. Для вставки пустой строки необходимо два раза поставить символ переноса строки (нажать на Enter)

Использование HTML

Редактор MarkDown так же может читать синтаксис HTML и работать вместе.
Например попробуйте написать **<h3>Hello World!</h3>**
Только удалите точки, они не нужны
