Selenium WebDriver

Данная статья относится к языку программирования Python

Что такое селениум? Selenium WebDriver – это программная библиотека для управления браузерами.
С Selenium вы можете имиторавать действия человека в интернете. Например открывать страницы сайтов, заполнять различные формы, авторизовываться.

В этой статье я расскажу вам, как установить Selenium на windows.


1) Сначало нужно скачать сам Selenium. В консоли прописать команду "pip install selenium"

2) Теперь нужно установить Selenium. В консоли прописать команду "pip.exe install selenium"

3) Последнее, что нужня для работы это Geckodriver .Geckodriver – это адаптер, преобразователь из протокола W3C WebDriver в протокол Marionette. Версию для вашей платформы вы можете скачать отсюда https://github.com/mozilla/geckodriver/releases/ . Скачанный файл просто разместите в той же дирректории, где написан ваш Selenium скрипт.

4) Давайте напишем первый скрипт для Selenium.

from selenium import webdriver

driver = webdriver.Firefox()

driver.get('https://www.vk.com')
login = driver.find_element_by_css_selector('#index_email')
login.send_keys('Ваш логин или почта')

password = driver.find_element_by_css_selector('#index_pass')
password.send_keys('Ваш пароль')

sumbit = driver.find_element_by_css_selector('#index_login_button')
sumbit.click()
